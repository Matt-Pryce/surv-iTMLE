# surv-iTMLE    

surv-iTMLE offers a solution for estimating treatment effect heterogenity using time-to-event data. It estimates the difference in conditonal survival probabilties under two exposures, and can handle both right censored as well as left truncated event times. 

Practical examples along with a tutorial are to be added shortly, but if you have any questions regarding its implementation in the meantime, please email me at: matthew.pryce@ucl.ac.uk.


**Supplementary materials**
The surv-iTMLE supplementary folder contains simulation scripts which relate to paper submissions.
